# ColorPlayer
Android color player

This is simple example of audio player.
Player has:
  - play/stop timer
  - 3 colors
  - add/remove playlists
  - sorting by folders, names, artists
  - equalizer
  - mini visualizer
  - find track by name
  - play in background
  - simple widget
